## Hypotheses (To Be Scored Against Evidence)

| ID | Hypothesis | Evidence Needed |
|---|---|---|
| H1 | Monetization / Pro tier | Strong retention; users hitting card limit or requesting scheduling |
| H2 | App Store submission (iOS) | App stable enough for public; desire to reach broader audience |
| H3 | Apple Watch companion | Users request wrist initiation; high timer completion rate |
| H4 | Android + Play Store | Inbound requests from Android users |
| H5 | Home screen widget | Evidence app-open friction causes abandonment |
| H6 | Siri / Shortcuts | Feature requests specifically mentioning voice/automation |
| H7 | LLM habit suggestions (opt-in) | Onboarding goal→action completion <70%; users struggle to create cards |
| H8 | iCloud backup | "Lost data on reinstall" in top 3 user complaints |
